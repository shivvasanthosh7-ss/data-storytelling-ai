# Data Storytelling AI PRO

An interactive, multi-feature data analysis web application built using **Python & Streamlit**.
This project transforms raw data into meaningful insights with dashboards, visualizations, and intelligent analysis — all in a clean, user-friendly interface.

---

## Features

### CSV Dashboard

* Upload CSV datasets
* KPI metrics (rows, columns, missing values)
* Data preview
* Automatic visualizations
* Correlation heatmaps

###Chat with Data

* Ask questions in natural language
* Example queries:

  * *average steps*
  * *max sleep_hours*
  * *total rows*
* Instant computed answers

###PDF Resume Analyzer

* Upload resume (PDF)
* Extract text automatically
* Detect technical skills
* Resume scoring system
* Suggestions for improvement

###Smart Visualizations

* Histograms for numeric columns
* Correlation heatmaps
* Data distribution insights

###Report Download

* Export summary reports
* Lightweight and fast

## Run Locally

```bash
git clone https://github.com/shivvasanthosh7-ss/data-storytelling-ai.git
cd data-storytelling-ai
pip install -r requirements.txt
streamlit run app.py
```
