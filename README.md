# 📊 Data Storytelling AI

An interactive web application that analyzes datasets and generates insights using Python and Streamlit.

---

## 🚀 Features
- Upload CSV datasets
- Data preview & summary statistics
- Missing value detection
- Data visualizations
- Correlation heatmap
- AI-powered insights 

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- OpenAI API

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
